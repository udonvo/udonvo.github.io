    Android & Web Development at Metron

    At Metron, I built a weather crowd-sourcing app that exploits an Android phone's native sensors (such as built-in thermometers, barometers, and GPS sensors) to report weather data to a remote database. The app supports bluetooth connectivity to pull environmental data from external weather sensors as well as capability for user-inputed data points.
